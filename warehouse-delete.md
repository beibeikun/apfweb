# 拍品删除

侧栏 **「仓库工具」→「拍品删除」**。

## 作用简述

在指定 **图片仓库** 中，根据输入的 **合同编号** 或 **拍品编号**，**永久删除** 对应的拍品 **ZIP**；识别规则与 **[拍品查询](/lot-query)** 一致：

- **按拍品**：若存在 `合同目录/拍品编号.zip`，则只删除该 ZIP（及可选的缩略图）。若删除后该合同目录下 **不再有任何** 拍品 ZIP，会 **删除整个合同文件夹**（其中非 ZIP 等条目也会一并删除，预览中会列出）。  
- **按合同**：若不存在单拍品 ZIP，但存在以输入为名的 **合同文件夹**，则删除该文件夹下 **全部** 拍品 ZIP（及对应缩略图），并 **移除整个合同目录**。

若该仓库在 [系统设置](/settings) 中启用了 **使用缩略图库**，会同时删除 `仓库根/thumbnail/拍品编号.jpg`（若存在）；未启用则 **不** 动 `thumbnail` 目录。

**此操作不可恢复**；务必先使用 **预览待删除** 核对清单，并经过两次确认对话框后再执行。

## 使用前准备

1. 在 [系统设置](/settings) 中配置可访问的 **图片仓库**，并确认 NAS/本地路径有效。  
2. 建议先在同一仓库使用 **[拍品查询](/lot-query)** 核对编号与路径是否符合预期。  
3. 确认没有其它流程仍依赖待删 ZIP 或合同目录内的文件。

## 操作步骤

1. **编号**：输入 **合同编号** 或 **拍品编号**（与拍品查询相同约定）。  
2. **图片仓库**：选择目标仓库。  
3. 点击 **预览待删除**：在 **删除预览** 区查看识别方式（按拍品 / 按合同）、将删除的 ZIP 与缩略图路径；若将删除整个合同文件夹，会列出目录内其它将一并删除的条目。  
4. 核对无误后点击 **执行删除**，在 **确认删除** 与 **最后确认** 中均选择继续。  
5. 完成后可按需再次预览以继续其它删除；成功或失败会写入 **总览** 中的最近任务记录。

## 常见问题

- **未匹配到拍品 ZIP 或合同文件夹**：输入或仓库结构与「拍品查询」规则不一致，请对照查询结果与仓库目录层级（`仓库根/合同/拍品.zip`、`仓库根/thumbnail/拍品.jpg`）。  
- **部分或全部删除失败**：预览区路径无法删除（权限、占用、已不存在等）时会列出错误；需在本机或 NAS 上排查后重试。  
- **缩略图未删除**：检查该仓库是否勾选 **使用缩略图库**；未启用时设计为不删除 `thumbnail` 下文件。  
- **误删合同目录**：当按拍品删除且合同内仅剩该 ZIP 时，会整目录删除；请始终以预览清单为准。
