# 备份归档

## 作用简述

扫描 **待归档图片文件夹**，按拍品编号分组，生成缩略图并将结果打包为 ZIP，写入所选 **图片仓库** 中对应的合同目录结构。

## 使用前准备

- 在 [系统设置](/settings) 中配置至少一个 **图片仓库**；若需「ZIP 内压缩」，还需配置 **压缩规则**。

## 操作步骤

1. 待归档图片文件夹：选择或拖入目录。
2. 目标图片仓库：从下拉选择（来自系统设置）。
3. 打包 ZIP 前压缩图片（可选）：勾选后按所选压缩规则重编码再写入 zip。
4. 归档冲突策略：
   - 跳过不处理：已存在同名 zip 则跳过。
   - 覆盖原有 ZIP：覆盖目标 zip。
5. 点击「开始一键归档」；完成后会提示完成。

## 常见问题

- 未选择有效待归档文件夹或仓库：检查路径与仓库列表。
- 已开启 ZIP 内压缩但未选规则：在系统设置中添加压缩规则后，在本页选择对应规则。
