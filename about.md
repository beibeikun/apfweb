# 关于 APF

AuctionPicFlow（APF）是一款面向拍卖行业的图片管理工作流工具，聚焦从图片整理、命名、压缩、校对到归档的全流程效率提升。

## 我们在做什么

- 为拍卖企业提供可落地的图片处理流程，减少重复人工操作。
- 通过规则化与批处理能力，降低图片资产管理过程中的出错率。
- 提供持续迭代的产品能力，并支持业务场景定制开发。

## 核心能力概览

- 全流程图片处理：整理、排序、改名、压缩、归档与校对。
- 多策略规则配置：命名、压缩、列映射等均可按业务调整。
- 跨平台支持：兼容 Windows 与 macOS。
- 性能优化：基于并发处理提升大批量任务处理效率。

## 服务与支持

- 提供全天候技术支持。
- 可按实际业务流程提供深度定制开发服务。
