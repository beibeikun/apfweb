## 1. 简介
“图片检查”是一个智能核对助手。它能将您的图片文件夹与 Excel 拍品清单进行自动比对，利用智能模型分析图片内容，帮您确认图片中的实物是否与清单记录的“名称”或“作者”一致，并快速标记出“匹配”、“存疑”或“不匹配”的结果。

## 2. 详细使用方法

### 第一步：准备工作
在开始检查前，请确保：
* **图片命名**：图片文件名应为拍品的编号（如 `A001.jpg`）。
* **清单准备**：您的 Excel 清单中应包含“编号”、“名称”和“作者”等列（具体的列名可以在系统设置中自定义）。

### 第二步：导入素材
1.  **选择图片文件夹**：点击“浏览”按钮或直接将文件夹**拖拽**到输入框中。
2.  **导入拍品清单**：点击“浏览”按钮选择对应的 Excel 文件（支持 `.xlsx` 和 `.xls` 格式）。

### 第三步：选择检查内容
在“检查项”区域，您可以根据需要勾选：
* **检查名称**：核对图片内容是否符合清单中的作品名称。
* **检查作者**：核对图片（如落款、题字）是否符合清单中的作者信息。
> *注意：您必须至少勾选其中一项才能开始工作。*

### 第四步：执行检查
1.  **开始检查**：点击“开始检查”按钮，系统将逐一识别图片并给出判定结论。
2.  **查看进度**：您可以通过进度条了解剩余工作量，并在下方的日志窗口查看正在处理的具体拍品。
3.  **结果预览**：处理完成后，中间的预览框会显示前 50 条结果的摘要，包括判定的理由（例如：为什么该图片被判定为“不匹配”）。

### 第五步：导出结果报告
1.  点击“**导出 Excel**”按钮。
2.  系统会在您预设的文件夹中创建一个新的子文件夹，并生成一份详细的《图片检查结果.xlsx》表格，方便您后续进行人工复核。

## 3. 常见问题与解决办法

| 现象 | 可能的原因 | 解决办法 |
| :--- | :--- | :--- |
| **无法点击“开始检查”** | 未选择图片文件夹、Excel 或未勾选检查项 | 请检查上方输入框是否都已填妥，并至少勾选“名称”或“作者”。 |
| **提示“模型配置未设置”** | 尚未配置智能分析助手 | 请前往“系统设置”页面，填写正确的模型连接信息。 |
| **识别结果全部显示失败** | Excel 中的列名与设置不匹配 | 请确保 Excel 中的列标题（如“名称”、“作者”）与您在系统设置中定义的名称完全一致。 |
| **导出时提示路径无效** | 未设置默认保存位置 | 请在“系统设置”中配置“默认输出根路径”，建议选择一个固定的工作文件夹。 |
| **检查过程中想要停止** | 发现文件选错或需要紧急处理其他事务 | 点击“**中止**”按钮，系统会安全停止并允许您导出已经完成的部分结果。 |
