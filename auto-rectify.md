# 辅助修正

> [!IMPORTANT] 版本时效性声明
> 当前文档对应版本：`v0.6.0`
> 
> 由于项目正处于**早期高频迭代阶段**，功能变更较为频繁，文档更新速度可能暂时无法完全同步。如发现内容缺失，请等待文档更新或联系开发者。

> [!WARNING] 配置声明
> 本模块的部分/全部功能依赖于 **大模型（LLM）连接**。请确保在使用前已在 [系统设置](/settings) 中完成模型连接设置。

侧栏 **「AI工具」→「辅助修正」**。

## 1. 简介
辅助修正可以将倾斜、变形或者拍摄角度不佳的图片（例如书画卷轴、纸质文档、海报等）重新校正为平整、方正的矩形图片，并支持批量自动化处理。

## 2. 详细使用方法

### 第一步：准备输入内容
1.  **选择路径**：在界面上方的“输入图片或文件夹”框中，点击【浏览】按钮选择您想要处理的单张图片或整个图片文件夹。您也可以直接将文件或文件夹从电脑桌面拖拽到输入框内。
2.  **配置模型**（可选）：如果您希望使用 AI 自动识别功能，请确保界面显示的“模型配置”为“已配置”状态。如果未配置，请先在系统设置中完成相关信息的填写。

### 第二步：选择修正模式
您可以根据图片的复杂程度选择不同的定位方式：
* **手动四点定位（推荐）**：点击【开始修正】后会弹出一个独立的大窗口，您可以精准地点击图片的四个角进行校正。
* **自动检测轮廓**：程序尝试自动寻找图片的边缘。如果背景简单、图片边缘清晰，这种模式最为快捷。

### 第三步：设置输出选项
* **输出对比图**：勾选后，程序会额外生成一张“处理前”与“处理后”的左右对比图，方便您检查效果。
* **边缘留白 (Padding)**：您可以设置一个数值，决定修正后的图片四周保留多少空白边缘（正值增加留白，负值裁掉边缘）。

### 第四步：开始执行
点击【开始修正】按钮，程序将开始工作。

## 3. 手动操作技巧（仅限“手动四点定位”模式）
在手动操作窗口中，您可以像使用专业修图软件一样轻松操作：

1.  **确定角点**：在大图上依次点击图片的四个顶点。
2.  **微调点位**：鼠标按住已经生成的点进行拖动。在拖动时，系统会自动开启**放大镜**功能，帮助您看清像素边缘。
3.  **快捷键操作**（提高效率）：
    * `Z`：显示或隐藏小放大镜。
    * `X`：撤销上一个点。
    * `C`：清空当前图片上的所有点。
    * `Q` / `E`：将图片逆时针或顺时针旋转 90 度。
    * `N`：跳过当前这张图，直接处理下一张。
    * `B`：返回处理上一张图。
    * `Enter`：确认当前的四个角点，并自动处理下一张图。
    * `↑` / `↓` / `,` / `.`：通过方向键和标点键快速微调上下左右的留白宽度。

## 4. 查看与保存结果
1.  **查看进度**：界面底部的进度条会显示当前批量任务的完成情况，日志框会记录处理成功的数量。
2.  **预览结果**：界面中央会显示最后一张处理完成的缩略图。
3.  **结果存放**：点击【打开输出目录】按钮，即可直接跳转到电脑中存放修正后图片的文件夹。
4.  **导出报告**：点击【导出 Excel】，系统会为您生成一份详细的清单，记录每张图片的修正参数和备注信息。

## 5. 常见问题与解决办法

| 现象 | 建议解决办法 |
| :--- | :--- |
| **图片读取失败** | 检查图片文件名是否包含极少数系统不支持的特殊符号，或检查图片文件是否已损坏。 |
| **自动检测模式效果不好** | 这通常是因为背景与图片颜色太接近。建议切换到“手动四点定位”模式手动指定四个角。 |
| **AI 功能无法启动** | 请检查网络连接是否正常，或在“系统设置”中检查模型配置是否填写正确。 |
| **输出图片边缘被裁掉了** | 请在“padding”设置处调大数值，或在手动模式下通过方向键增加边缘宽度。 |
