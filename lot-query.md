# 拍品查询

## 作用简述

在选定图片仓库中输入合同编号或拍品编号，应用自动判断查询模式：

- **按拍品**：若存在对应拍品 zip，显示 zip 路径、创建时间等信息，并在有缩略图时展示缩略图。
- **按合同**：若无单拍品 zip 但存在同名合同文件夹，则列出该合同下全部拍品 zip 及创建时间。

## 操作步骤

1. 编号：输入合同号或拍品号。
2. 图片仓库：选择已在 [系统设置](/settings) 中配置的仓库。
3. 点击「查询」；结果在查询结果文本区显示。

## 说明

若既找不到 `合同目录/拍品编号.zip`，也找不到以输入为名的合同文件夹，结果区会说明可能的路径约定，便于核对仓库结构是否与预期一致。

## 常见问题

- 提示选择仓库：在系统设置中添加图片仓库。
- 路径无效：检查 NAS 挂载与仓库路径是否可访问。
