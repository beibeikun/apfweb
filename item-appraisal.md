# 拍品鉴定

> [!IMPORTANT] 版本时效性声明

>

> [!WARNING] 配置声明

侧栏 **「AI工具」→「拍品鉴定」**。

## 1. 简介
“拍品鉴定”模块是一款智能化的信息提取工具。它能够自动识别图片中的艺术品或收藏品信息，通过智能分析，自动整理出作者、作品名称、分类、质地以及材质等关键数据，并最终汇总成整齐的 Excel 表格。

## 2. 详细使用方法

### 第一步：准备图片文件
为了确保系统能正确识别每一件拍品，请在处理前完成以下准备：
* **规范命名**：请将图片文件的名称修改为对应的“**拍品编号**”（例如：`A001.jpg`）。
* **整理存放**：将所有需要鉴定的图片放在同一个文件夹内。

### 第二步：导入文件夹
1.  打开“拍品鉴定”界面。
2.  在“**图片文件夹**”栏目中，点击右侧的“**浏览**”按钮选择您的图片文件夹；或者直接将文件夹从电脑中**拖拽**到输入框内。

### 第三步：检查配置状态
在界面中部，您会看到一行提示文字（当前模型配置...）。请确保：
* **模型配置**已完成（显示“已配置”）。
* **规则预设**已加载（显示分类、质地等项数）。
> *注：如果提示“未配置”，请先前往“系统设置”完成相关连接设置。*

### 第四步：开始自动鉴定
1.  点击“**开始鉴定**”按钮。
2.  **观察进度**：下方的进度条会显示当前的处理进程，日志窗口会实时滚动显示正在处理的文件名。
3.  **查看初步结果**：处理完成后，界面中间的列表会展示前 50 条鉴定结果的摘要（包括编号、作者、作品名等），供您快速预览。

### 第五步：导出 Excel 表格
1.  鉴定完成后，点击“**导出 Excel**”按钮。
2.  系统会自动在您预设的存放位置创建一个以“**当前日期时间_拍品鉴定**”命名的文件夹，并将详细的表格保存其中。
3.  导出成功后，系统会弹出提示并播放完成音。

## 3. 常见问题与解决办法

| 现象 | 可能的原因 | 解决办法 |
| :--- | :--- | :--- |
| **点击“开始”后提示“请选择有效的文件夹”** | 输入框为空或路径被手动删改过 | 请重新点击“浏览”按钮选择正确的图片目录。 |
| **提示“请先配置模型信息”** | 尚未连接到智能服务 | 请点击软件的“系统设置”，填写连接地址和授权密钥。 |
| **鉴定出的分类不是我想要的** | 结果超出了允许的范围 | 系统会严格按照设置中的“允许值列表”进行分类。如需更改，请在设置中修改“鉴定规则预设”。 |
| **点击“导出”时报错** | 未设置默认保存位置 | 请在“系统设置”中指定一个“默认输出根路径”。 |
| **想要中途停止** | 处理时间过长或发现图片选错 | 点击“**中止**”按钮，系统会停止后续处理并保留已经完成的部分结果。 |
