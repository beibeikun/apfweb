# 选中改名

> [!IMPORTANT] 版本时效性声明
> 当前文档对应版本：`v0.6.0`
> 
> 由于项目正处于**早期高频迭代阶段**，功能变更较为频繁，文档更新速度可能暂时无法完全同步。如发现内容缺失，请等待文档更新或联系开发者。

侧栏 **「核心工具」→「选中改名」**。

## 作用简述

在 **访达（macOS）或资源管理器（Windows）** 中，于 **同一文件夹内** 多选若干 **JPEG**（`.jpg` / `.jpeg`），回到本应用填写 **合同编号** 与 **序号**，合成 **拍品编号**（如 `A123-001`），再按所选 **输出文件名格式** 将选中文件 **原地批量重命名** 为该拍品的主图与细节序号。

本工具提供 **标准改名** 与 **快速改名（悬浮窗连续处理）** 两种模式。可选多种 **排序方式**（含「与整理排序一致」、按文件名/修改时间等），以及是否 **使用主图格式**。成功执行后可 **撤销** 最近一次改名（若目标文件仍完整存在）。

## 使用前准备

1.  在 [系统设置](https://www.google.com/search?q=/settings) 的 **命名规则管理** 中维护 **输出文件名格式**，保存后在本页下拉中选择要用的规则。
2.  在访达或资源管理器中打开目标目录，**只选中该目录下的 JPEG**（不要跨文件夹多选）。
3.  **macOS**：保持 **访达** 为前台或确保其选中项有效；应用通过系统接口读取当前选中路径。
4.  **Windows**：通过资源管理器 COM 读取选中项；若多个窗口均有选中，行为以实现为准，建议只在一个 Explorer 窗口中选中目标文件后再进行操作。

## 基础参数配置

在执行任何改名模式前，请先在主界面确认以下配置：

1.  **序号位数（补零）**：与填写的序号组合成拍品号里连字符后的部分（如序号 `1`、位数 `3` → `…-001`）。**注意**：这只影响拍品编号中的这一段，与命名规则里「细节序号」的补零位数无关。
2.  **输出文件名格式**：从下拉选择已在系统设置中配置好的规则。
3.  **排序方式**：按需选择；若要智能识别主图并让细节图按序号排列，选 **与整理排序一致（主图优先、细节序号）**。
4.  **使用主图格式**：勾选时，首张为 `拍品编号.jpg`（或所选规则中的主图模板），其余为 `拍品编号 (n).jpg`；取消勾选则该批全部从 `(1)` 起编号，无单独无括号的主图名。

## 操作步骤

### 模式一：标准改名

适合单次处理或需要频繁调整参数的场景。

1.  在访达/资源管理器中 **选中** 要改名的 `.jpg` / `.jpeg`（须位于同一父文件夹）。
2.  切换到本应用，填写 **合同编号**（须以 **英文字母开头**，且仅含字母与数字，如 `A123`）与 **序号**（仅数字）。
3.  点击 **「改名」**（或在输入框按回车）；下方日志会显示拍品编号与处理进度。若文件名已与目标一致，会提示无需改名。

### 模式二：快速改名（推荐多组连扫）

适合需要连续为多组拍品改名的场景，极大减少鼠标点击与窗口切换。

1.  配置好基础参数后，点击 **「开始快速改名」**。
2.  主界面将自动最小化，屏幕中央会弹出一个置顶的 **「快速改名」悬浮小窗**。
3.  在访达/资源管理器中 **选中** 第一组拍品图片。
4.  在悬浮窗中输入 **合同编号**，按 `Enter`（回车），光标会自动跳至序号框。
5.  输入 **序号**，按 `Enter` 立即执行选区改名。
6.  **循环操作**：改名成功后，悬浮窗会 **自动保留当前的合同编号**，并清空序号框、重新聚焦。你只需在文件管理器中选中下一组图片，直接在键盘上盲打新序号并回车即可。
7.  关闭悬浮小窗即可恢复主界面。

### 撤销操作

需要恢复时，在 **未进行新的改名** 前可点 **「撤销」**。系统会将上一批被改名的文件恢复为原始文件名（若磁盘上目标文件已被移动或删除则可能失败）。撤销仅对最后一次操作有效。

## 常见问题

  - **未选中任何 JPEG / 未在选中项中找到 JPEG**：请确认访达或资源管理器中选中的是图片文件，且扩展名为 `.jpg` 或 `.jpeg`。
  - **请仅在同一文件夹内选中文件**：不能同时选中多个目录下的文件。
  - **合同编号 / 序号校验失败**：合同编号不能为空，须字母开头且仅字母数字；序号须为非空数字串，不能含空格。
  - **目标已存在，无法完成改名**：目标文件名已被 **其他未参与本批次的文件** 占用，需先手动处理重名。
  - **当前系统不支持从文件管理器读取选中项**：Linux 等未实现该能力；请在 macOS 或 Windows 上使用。
  - **输出文件名格式为空**：请先在系统设置 **命名规则管理** 中添加规则并保存，再回到本页选择。
