# 选中改名

侧栏 **「核心工具」→「选中改名」**。

## 作用简述

在 **访达（macOS）或资源管理器（Windows）** 中，于 **同一文件夹内** 多选若干 **JPEG**（`.jpg` / `.jpeg`），回到本应用填写 **合同编号** 与 **序号**，合成 **拍品编号**（如 `A123-001`），再按所选 **输出文件名格式**（与 [系统设置](/settings) 中 **命名规则管理** 的规则一致，与改名压缩共用）将选中文件 **原地批量重命名** 为该拍品的主图与细节序号。

可选 **排序方式**（含与 [整理排序](/organize-sort) 一致的「主图优先、细节序号」、按文件名/修改时间、或保持访达/资源管理器中的选中顺序）。可选是否 **使用主图格式**（首张为 `拍品编号.jpg`，其余为 `拍品编号 (n).jpg`；关闭则全部从 `拍品编号 (1).jpg` 起连续编号）。成功执行后可 **撤销** 最近一次改名（若目标文件仍完整存在）。

## 使用前准备

1. 在 [系统设置](/settings) 的 **命名规则管理** 中维护 **输出文件名格式**，保存后在本页 **输出文件名格式** 下拉中选择要用的规则（可与系统默认规则一致）。  
2. 在访达或资源管理器中打开目标目录，**只选中该目录下的 JPEG**（不要跨文件夹多选）。  
3. **macOS**：保持 **访达** 为前台或确保其选中项有效；应用通过系统接口读取当前选中路径。  
4. **Windows**：通过资源管理器 COM 读取选中项；若多个窗口均有选中，行为以实现为准，建议只在一个 Explorer 窗口中选中目标文件后再点改名。

## 操作步骤

1. 在访达/资源管理器中 **选中** 要改名的 `.jpg` / `.jpeg`（须位于 **同一父文件夹**）。  
2. 切换到本应用的 **选中改名** 页。  
3. **合同编号**：须以 **英文字母开头**，且仅含英文字母与数字（如 `A123`）。  
4. **序号**：仅数字；与 **序号位数（补零）** 组合成拍品号里连字符后的部分（如序号 `1`、位数 `3` → `…-001`）。**序号位数（补零）** 只影响拍品编号中的这一段，与命名规则里「细节序号」位数不是同一项。  
5. **输出文件名格式**：从下拉选择已在系统设置中配置好的规则。  
6. **排序方式**：按需选择；若要与整理排序逻辑一致，选 **与整理排序一致（主图优先、细节序号）**。  
7. **使用主图格式**：与整理排序含义相同；取消则该批全部从 `(1)` 起编号，无单独无括号主图名。  
8. 点击 **改名**；日志会显示拍品编号与进度。若文件名已与目标一致，会提示无需改名。  
9. 需要恢复时，在 **未进行新的改名** 前可点 **撤销**（若磁盘上目标文件已被移动或删除则可能失败）。

## 常见问题

- **未选中任何 JPEG / 未在选中项中找到 JPEG**：请确认访达或资源管理器中选中的是图片文件，且扩展名为 `.jpg` 或 `.jpeg`。  
- **请仅在同一文件夹内选中文件**：不能同时选中多个目录下的文件。  
- **合同编号 / 序号校验失败**：合同编号不能为空，须字母开头且仅字母数字；序号须为非空数字串，不能含空格。  
- **目标已存在，无法完成改名**：目标文件名已被 **其他未参与本批次的文件** 占用，需先手动处理重名。  
- **当前系统不支持从文件管理器读取选中项**：Linux 等未实现该能力；请在 macOS 或 Windows 上使用。  
- **输出文件名格式为空**：请先在系统设置 **命名规则管理** 中添加规则并保存，再回到本页选择。
