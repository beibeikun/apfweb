# Auction-Pic-Flow 激活密钥使用说明（RSA）

本文件说明如何给用户生成并发放激活密钥，以及用户如何完成激活。

## 1. 激活密钥是什么格式？

激活密钥由两段 Base64 字符串拼接而成，格式如下：

`数据Base64.签名Base64`

## 2. 用户怎么获取机器码并激活？

用户只需要完成两步：复制提供机器码、粘贴发卡方提供的激活密钥到软件。

### 2.1 首次启动（或激活失败后）

1. 用户启动软件。
2. 程序会弹出“激活验证”窗口，窗口中会显示“本机机器码（只读）”与激活密钥输入框。
3. 用户点击“复制机器码”，把机器码内容发给你。

### 2.2 输入激活密钥

1. 发卡方生成好激活密钥后，把整段激活密钥字符串发给用户。
2. 用户把激活密钥粘贴到“激活验证”窗口的输入框。
3. 点击确认后，如果校验通过，软件会进入主界面。

## 3. 删除当前激活密钥

用户可以在软件内删除激活密钥并立即退出，避免继续以旧激活状态运行。

操作路径：

1. 打开“系统设置”
2. 找到“许可证/激活密钥管理”
3. 点击“删除当前激活密钥”
4. 需要二次确认
5. 删除并立刻关闭软件

下次启动将再次进入激活流程（需要重新输入激活密钥）。

## 4. 常见问题

1. **激活失败提示“验签失败/不匹配当前设备”**
   - 多半原因： `machine_id` 复制有误。
2. **启动提示无法获取网络时间**
   - 启动阶段会请求服务器交换数据；需要网络可用。
