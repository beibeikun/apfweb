# 系统设置

> [!IMPORTANT] 版本时效性声明
> 当前文档对应版本：`v0.6.0`
> 
> 由于项目正处于**早期高频迭代阶段**，功能变更较为频繁，文档更新速度可能暂时无法完全同步。如发现内容缺失，请等待文档更新或联系开发者。

侧栏底部点击 **「系统设置」**。

保存后，各业务页会从磁盘重新加载配置（无需重启应用，部分界面主题/语言切换会立即生效）。

## 常规设置

1. **界面主题与语言**：支持浅色、深色、跟随系统设置，以及多语言（简体中文/日本語等）的界面语言切换。
2. **任务提示音**：开启后，耗时任务完成时会播放提示音。
3. **默认输出根路径（建议优先配置）**：
   「改名压缩」「提取主图」「图片校对」导出、「仓库分析」导出等均依赖此目录。请选择可写、空间充足的上拍输出根目录。

## Bark 推送

支持在任务完成时通过 Bark 发送移动端推送通知。
- **任务分类**：勾选具体的任务分类（如改名压缩、仓库分析等）即可定制推送范围，与总览页最近任务分类一致。
- **连接配置**：需填写 Bark 服务器地址（默认例如 `https://api.day.app`）和 Bark 应用中的设备 Key。支持点击“发送测试推送”验证连接。

## Excel 列规则管理

配置读取外部 Excel 表格时使用的表头列名，需与真实表格表头完全一致：
- **拍品编号列**：换号表、校对、主图筛选等读取的列名（例：`拍品编号`）。
- **Lot 列**：上拍输出命名中的 `{lot}` 来源（例：`Lot号`）。
- **名称列** 与 **作者列**：用于特定业务读取对应的拍品信息（古董等类别作者可留空）。

## 拍品鉴定规则预设

为大模型鉴定提供允许的分类选项（支持换行录入，一行一个）：
- **拍品分类**（例：书画、瓷器、玉器、未知）
- **质地款式**（例：立轴、手卷、镜心、未知）
- **材质形式**（例：纸本设色、绢本水墨、青花瓷、未知）

> **提示**：模型输出的“拍品分类/质地款式/材质形式”只能从此处允许的值中选择；建议务必在每一项中保留兜底选项「未知」。

## 模型连接设置

配置用于智能提取、鉴定的 LLM（大语言模型）API：
- **基础连接**：Base URL（兼容 OpenAI 格式）与 API Key。
- **模型参数**：模型名称（如 `gpt-4o-mini`）与请求超时时间（秒）。
- **高级推理**：
  - 支持请求 `thinking` 模式（若接口支持，仅使用最终回答，不展示思考过程）。
  - 可调整 Reasoning 强度（主要针对 Ollama 等支持 `reasoning_effort` 的模型）。

## 图片仓库工具

- 填写「仓库名称」与「仓库路径」，点击「添加/更新仓库」。
- **使用缩略图库**：勾选后可在仓库根目录维护 `thumbnail/` 文件夹，提升预览加载速度。
- 选中列表中已有仓库后，可用「设为默认」；默认仓库会在各页面的仓库下拉框中优先被选中。
- 路径一般配置为 NAS 或本地挂载的归档根目录。

## 压缩规则管理

维护多条图片压缩规则：
- 可分别设置 **最长边 (px)** 与 **最大体积 (KB)**。两项可只填其一，另一项选「留空」表示不按该项限制。
- 选中后可用「设为默认」，各业务页的压缩规则下拉会优先选中该条。

## 命名规则管理

「上拍输出文件名格式」使用的规则在此维护：
- **序号位数**：指定连续编号的补零位数（如填 2 则输出 `01`, `02` 等）。
- 占位符说明：`{lot}` 代表 Lot 号，`{seq}` 代表序号。
- **统一命名模板**：若填写（例：`{lot}_{seq}.jpg`），则主图与细节均按该模板连续编号。
- **主图/细节图模板**：若统一模板留空，则分别使用「主图模板」（需含 `{lot}`）与「细节图模板」（需含 `{lot}` 与 `{seq}`）。

> **注意**：系统中至少需要保留一条命名规则；同样支持「设为默认」。

## 配置备份与迁移

将当前所有的系统设置与本机已缓存的仓库分析结果导出为单个 JSON 文件。
- **导出配置**：用于备份或设备间迁移。
- **导入配置**：导入后将**覆盖**当前设置。若导入文件内含仓库分析字段，则替换本机缓存；若该字段为 null 则清除缓存；若省略则保留本机现有缓存。

## 许可证/激活码管理

用于查看或管理软件授权：
- 展示当前软件的 **激活状态** 与 **有效期至**（基于本地时间判断）。
- **删除当前激活码**：如需解绑或更换激活码，可在此处删除。删除操作将立即关闭当前软件。

## 保存与关于

- 点击 **「保存所有设置」** 立即将变动写入本机配置文件。
- 点击 **「关于」** 查看软件版本号与相关开发信息。
