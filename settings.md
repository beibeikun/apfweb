# 系统设置

侧栏底部「系统设置」。保存后，各业务页会从磁盘重新加载配置（无需重启应用）。

## 建议优先配置

1. **默认输出根路径**  
   「改名压缩」「提取主图」「图片校对」导出、「仓库分析」导出等均依赖此目录。请选可写、空间充足的上拍输出根目录。

2. **图片仓库**（折叠区「图片仓库管理」）
   - 填写「仓库名称」与「仓库路径」，点「添加/更新仓库」。
   - 在列表中选中一条后，可用「设为默认」；默认仓库会在各页下拉框中优先选中。
   - 路径一般为 NAS 或本地挂载的归档根目录（与「拍品查询」「备份归档」等约定一致）。

3. **Excel 列名**
   - 「拍品编号列」：换号表、校对、主图筛选等读取的列名（默认可为 `拍品编号`）。
   - 「Lot 列」：上拍输出命名中的 `{lot}` 来源（默认可为 `Lot号`）。
   - 需与真实表格表头一致。

## 压缩规则

在「压缩规则管理」中维护多条规则（最长边、最大体积 KB；可只填其一，另一项留空表示不限制）。

设为默认后，各页压缩规则下拉会优先选中该条。

## 改名压缩输出命名

「上拍输出文件名格式」使用的规则在此维护：

- 占位符：`{lot}` 为 Lot 号，`{seq}` 为序号。
- **统一命名模板**：若填写，则主图与细节均按该模板连续编号（须同时含 `{lot}` 与 `{seq}`）。
- 若统一模板留空，则分别使用「主图模板」与「细节图模板」（细节须含 `{lot}` 与 `{seq}`）。

至少保留一条命名规则；可用「设为默认」指定默认项。

## 界面主题

可选浅色、深色或跟随系统；变更会立即应用到当前会话。

## 保存

点击「保存所有设置」写入本机配置文件；路径说明见项目根目录 README 中的「配置文件位置」。
